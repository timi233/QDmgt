1. 系统简介

渠道管理系统是一款专为渠道销售团队设计的一站式管理平台，帮助企业高效管理分销商网络、追踪销售任务、监控渠道健康状态，实现渠道业务的数字化运营。

1.1 系统特点

  • 一站式管理 - 分销商信息、任务、拜访、培训等功能集中管理
  • 健康度评估 - 智能评估渠道伙伴健康状态，预警潜在风险
  • 工作流协同 - 任务分配、进度跟踪、团队协作一体化
  • 数据驱动 - 多维度数据统计与分析，辅助业务决策
  • 飞书集成 - 支持飞书单点登录，与企业办公系统无缝对接

1.2 用户角色

角色 说明 权限范围
Admin 管理员 系统管理员 所有功能 + 用户管理
Leader 负责人 团队负责人 团队数据查看 + 目标设定 + 数据看板
Sales 销售 一线销售人员 个人分销商和任务管理

2. 登录与注册

2.1 账号密码登录

1
访问系统登录页面，输入注册邮箱地址
2
输入密码（至少8位字符）
3
点击"登录"按钮进入系统

2.2 飞书一键登录

如果您的企业已配置飞书集成，可点击"飞书一键登录"按钮，系统将跳转至飞书授权页面。授权后自动完成登录并跳转到工作台。

提示
首次使用飞书登录的用户，系统将自动创建账号并同步飞书用户信息。

2.3 新用户注册

1
点击登录页"立即注册"链接
2
填写用户名、邮箱、姓名（选填）、密码
3
确认密码后点击"注册"
4
注册成功后自动登录并进入工作台

3. 工作台

工作台是用户登录后的首页，展示个人任务看板和工作统计数据。

3.1 任务看板

任务按紧急程度分为三栏展示：

  • 紧急任务 - 已逾期、优先级为紧急、或距离截止时间不足4小时的任务
  • 今日任务 - 截止日期为今天的待处理任务
  • 即将到期 - 其他未完成的任务

3.2 统计卡片

指标 说明
总任务数 当前所有任务总数
已完成 状态为已完成的任务数量
已逾期 超过截止时间未完成的任务数量
完成率 已完成任务占总任务的百分比

3.3 创建任务

点击右上角"新建任务"按钮，在弹窗中填写：

  • 任务标题 - 2-100个字符
  • 任务描述 - 至少10个字符，详细说明任务内容
  • 关联经销商 - 选择相关的分销商（可选）
  • 优先级 - 低/中/高/紧急
  • 截止时间 - 选择任务截止日期和时间

4. 分销商管理

分销商管理模块用于维护渠道伙伴的基本信息、分类标签和业务数据。

4.1 分销商列表

列表页展示所有分销商，支持以下操作：

  • 搜索 - 按名称、联系人、电话等关键词搜索
  • 筛选 - 按区域、合作等级、渠道类型等条件筛选
  • 排序 - 按创建时间、健康度评分等字段排序

4.2 分销商分类

合作等级

等级 说明
铂金 最高级别战略合作伙伴
金牌 核心合作伙伴
银牌 重要合作伙伴
铜牌 标准合作伙伴

渠道层级

层级 说明
战略级 战略合作伙伴，享有最高支持级别
核心级 核心业务伙伴
标准级 标准授权伙伴
发展级 新发展或培育中的伙伴

伙伴类型

  • ISV - 独立软件开发商
  • SI - 系统集成商
  • VAR - 增值分销商
  • 代理商 - 区域代理
  • 经销商 - 产品经销商

4.3 新建分销商

点击"新建分销商"按钮，填写以下信息：

基本信息
  • 分销商名称（必填）
  • 所在区域（省/市/区三级选择）
  • 联系人姓名
  • 联系电话
渠道分类
  • 合作等级
  • 渠道层级
  • 伙伴类型
  • 渠道类型（基础盘/高价值/待开发等）
业务信息
  • 授信额度
  • 季度目标
  • 产品线
  • 行业聚焦
  • 服务区域

5. 拜访记录

拜访记录模块用于记录与分销商的沟通拜访情况，追踪客户关系维护进展。

5.1 拜访类型

类型 说明
现场拜访 上门拜访客户
线上会议 视频/音频会议沟通
电话沟通 电话拜访
会议活动 展会、研讨会等活动场合

5.2 记录拜访

创建拜访记录需填写：

  • 拜访对象 - 选择分销商
  • 拜访日期 - 拜访时间
  • 拜访类型 - 现场/线上/电话/会议
  • 拜访目的 - 本次拜访的主要目的
  • 参与人员 - 双方参与人员名单
  • 讨论要点 - 会议/沟通主要内容
  • 客户反馈 - 客户的意见和建议
  • 后续行动 - 需要跟进的事项
  • 满意度评分 - 1-5分评价客户满意度
最佳实践
每次拜访后及时记录，确保信息完整准确。满意度评分有助于评估渠道健康状况。

6. 健康度管理

健康度管理通过多维度指标评估分销商的业务健康状态，帮助识别风险和机会。

6.1 评分维度

业绩维度（30分）
  • 营收增长率
  • 目标达成率
  • 订单频率
参与度维度（25分）
  • 产品线覆盖数
  • 培训参与度
活跃度维度（25分）
  • 最近订单天数
  • 最近联系天数
  • 响应率
满意度维度（20分）
  • 平均满意度评分
  • 投诉数量

6.2 健康状态

状态 评分范围 说明
健康 80-100分 伙伴状态良好，保持现有合作
预警 60-79分 存在潜在风险，需要关注
风险 40-59分 明显问题，需要及时干预
休眠 0-39分 长期不活跃，需要激活或清理

6.3 风险预警

系统会根据健康度评分自动生成预警和改进建议，包括：

  • 长时间未联系提醒
  • 订单下降预警
  • 满意度下降提醒
  • 认证即将过期提醒

7. 工作计划

工作计划模块支持月度工作计划的制定、执行和复盘。

7.1 月度计划

每月初为每个重点分销商制定工作计划，包括：

  • 机会来源 - 本月业务机会的来源渠道
  • 项目管理 - 重点项目的推进计划
  • 渠道动作 - 计划开展的渠道活动

7.2 周度复盘

每周对月度计划进行复盘更新：

  • 进展分析 - 本周工作进展情况
  • 障碍识别 - 遇到的困难和阻碍
  • 计划调整 - 下周工作计划的调整

7.3 计划状态

状态 说明
计划中 月初制定，尚未开始执行
执行中 正在按计划执行
已完成 月底完成结案

8. 培训管理

培训管理模块用于组织和跟踪面向渠道伙伴的培训活动。

8.1 培训类型

类型 说明
产品培训 产品功能、特性、使用方法培训
销售培训 销售技巧、话术、方案培训
技术培训 技术方案、实施、运维培训
认证培训 认证考试相关培训

8.2 培训形式

  • 线上培训 - 网络直播或视频会议
  • 线下培训 - 现场集中培训
  • 视频课程 - 录播视频自主学习

8.3 参训管理

培训组织者可以：

  • 创建培训活动，设置时间、地点、名额
  • 管理参训人员报名
  • 记录出勤情况
  • 录入考核成绩
  • 颁发培训证书

9. 资料库

资料库集中管理面向渠道伙伴的各类资料文档。

9.1 资料分类

分类 说明
产品文档 产品手册、技术规格、使用指南
销售工具包 销售话术、报价模板、方案模板
成功案例 客户案例、行业解决方案
政策文件 渠道政策、合作协议模板
视频资料 产品演示、培训录像
白皮书 行业报告、技术白皮书

9.2 访问权限

资料可设置访问权限，根据分销商等级控制可见范围：

  • 全部可见 - 所有分销商可访问
  • 铂金专属 - 仅铂金级伙伴可访问
  • 金牌及以上 - 金牌、铂金级伙伴可访问
  • 银牌及以上 - 银牌及以上伙伴可访问

10. 支持工单

支持工单系统用于处理渠道伙伴的问题反馈和支持请求。

10.1 工单类型

类型 说明
技术支持 产品技术问题、故障处理
销售支持 报价、方案、项目支持
政策咨询 渠道政策、合作模式咨询
账务问题 结算、发票、信用额度问题
其他 其他类型问题

10.2 优先级

优先级 响应时间 适用场景
紧急 2小时 生产故障、重大客户问题
高 4小时 影响业务的问题
普通 24小时 一般咨询和问题
低 48小时 建议、非紧急需求

10.3 工单状态

  • 待处理 - 新建工单，等待分配
  • 已分配 - 已分配处理人
  • 处理中 - 正在处理
  • 待反馈 - 等待客户反馈
  • 已解决 - 问题已解决
  • 已关闭 - 工单关闭归档

11. 认证管理

认证管理模块用于追踪渠道伙伴的资质认证状态。

11.1 认证类型

类型 说明
产品认证 特定产品线的销售/实施资质
销售认证 销售能力资质认证
技术认证 技术实施能力认证
合规认证 行业合规资质认证

11.2 认证级别

  • 金牌认证 - 最高级别，享有最优政策
  • 银牌认证 - 中级认证
  • 铜牌认证 - 基础认证

11.3 认证验证

系统提供认证验证功能，通过输入验证码可查询认证的真实性和有效期。

注意
认证到期前30天系统会自动提醒，请及时安排续证。

12. 目标管理

Leader Admin

目标管理模块用于设定和追踪渠道业务目标。

12.1 目标周期

  • 年度目标 - 全年业务总目标
  • 季度目标 - 分季度目标
  • 月度目标 - 分月度目标

12.2 目标指标

指标 说明
新签目标 新签约伙伴数量目标
核心机会 核心业务机会金额目标
核心营收 核心业务收入目标
高价值机会 高价值项目机会目标
高价值营收 高价值项目收入目标

12.3 目标追踪

系统自动统计目标完成情况，支持：

  • 实时查看各指标完成进度
  • 目标达成率可视化展示
  • 历史目标数据对比

  • 人员维度 - 按销售人员查看

15. 典型工作流程

15.1 新分销商入驻流程

流程步骤
1
创建分销商 - 在分销商管理中新建，填写基本信息和分类
2
安排首次拜访 - 创建拜访记录，记录首次沟通情况
3
制定工作计划 - 为该分销商创建月度工作计划
4
安排培训 - 邀请参加产品培训
5
申请认证 - 完成培训后申请相关认证

15.2 日常工作流程

每日工作
1
查看工作台 - 登录系统，查看今日任务和紧急事项
2
处理任务 - 按优先级处理待办任务，更新任务状态
3
记录拜访 - 完成客户拜访后及时记录
4
处理工单 - 响应分配给自己的支持工单

15.3 月度工作流程

月度循环
1
月初计划 - 制定当月工作计划，设定重点分销商工作方向
2
周度复盘 - 每周更新计划执行进展，识别问题并调整
3
健康度检查 - 关注预警和风险状态的分销商
4
月末结案 - 完成当月工作计划，总结经验教训

15.4 问题处理流程

工单处理
1
创建工单 - 分销商提出问题，创建支持工单
2
分配处理 - 根据问题类型分配给相应人员
3
处理跟进 - 处理人员跟进问题，添加处理记录
4
解决关闭 - 问题解决后关闭工单，记录满意度
需要帮助？
如在使用过程中遇到问题，请联系系统管理员或查阅在线帮助文档。
