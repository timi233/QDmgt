# User Guide - Channel Management System

## Getting Started

### Login

1. Open browser and navigate to system URL
2. Enter username and password
3. Click "Login" button

### First Time Setup

Contact system administrator to:
- Create your account
- Assign your role (Sales / Leader)
- Set initial password

## For Sales Users

### Dashboard Overview

After login, you will see the Sales Workbench with:
- Task priority list (urgent tasks first)
- Quick actions
- Today's summary

### Managing Dealers

#### View Dealer List
1. Click "Dealers" in sidebar
2. Use filters to search by name, region, or status
3. Click dealer name to view details

#### Add New Dealer
1. Click "New Dealer" button
2. Complete 3-step form:
   - **Step 1**: Basic Info (name, contact, phone) - Required
   - **Step 2**: Business Details (address, region, scale) - Required
   - **Step 3**: Additional Info (notes, tags) - Optional
3. Click "Save Draft" to save progress
4. Click "Submit" when complete

#### Edit Dealer
1. Open dealer details
2. Click "Edit" button
3. Modify information
4. Click "Save"

#### Form Validation Rules
- Company name: 2-100 characters, required
- Contact person: 2-50 characters, required
- Phone: Valid mobile/landline format, required
- Email: Valid email format (if provided)
- Business scale: Select from list

### Managing Tasks

#### View Your Tasks
1. Click "Tasks" in sidebar
2. Tasks are sorted by priority and due date
3. Overdue tasks shown in red

#### Task Status Flow
- **Pending**: New task, not started
- **In Progress**: Currently working on
- **Completed**: Work finished
- **Overdue**: Past due date (auto-marked)

#### Update Task Status
1. Open task details
2. Click status button to advance:
   - Pending -> In Progress -> Completed
3. Add notes when completing

#### Task Collaboration
- **Add Collaborator**: Share task with team member
- **Add Comment**: Discuss task progress
- **Transfer Task**: Reassign to another sales person

### Tips for Sales

1. **Check tasks daily** - Start with overdue tasks
2. **Update dealer info promptly** - Keep contact details current
3. **Use notes field** - Record important conversation details
4. **Save drafts often** - Don't lose your work

## For Leader Users

### Dashboard Overview

After login, you will see the Leader Dashboard with:
- KPI summary cards
- Data tables with filters
- Team performance overview

### KPI Cards

- **Total Dealers**: All dealers in system
- **New This Month**: Dealers added this month
- **Conversion Rate**: Deals closed / Total dealers
- **Regional Distribution**: Breakdown by region

### Data Analysis

#### Filter Data
1. Use date range picker for time period
2. Select sales person to view individual performance
3. Choose region for geographic analysis

#### Export Data
1. Set desired filters
2. Click "Export Excel" button
3. File downloads automatically

### Task Management

#### View Team Tasks
1. Click "Tasks" in sidebar
2. See all team tasks
3. Filter by assignee, status, or due date

#### Assign New Task
1. Click "New Task" button
2. Select assignee
3. Set due date and priority
4. Add task description
5. Click "Create"

#### Monitor Progress
- Red: Overdue
- Yellow: Due soon (within 3 days)
- Green: On track

### Tips for Leaders

1. **Review dashboard daily** - Monitor KPIs and trends
2. **Check overdue tasks** - Follow up with team
3. **Export monthly reports** - Track team performance
4. **Use filters** - Focus on specific areas or team members

## Common Operations

### Change Password

1. Click your name in top right
2. Select "Settings"
3. Enter current password
4. Enter new password twice
5. Click "Save"

### Logout

1. Click your name in top right
2. Click "Logout"

### Get Help

For technical issues, contact system administrator.

## Frequently Asked Questions

**Q: I forgot my password**
A: Contact system administrator to reset

**Q: I can't see a dealer I created**
A: Check your filters - you may have search criteria set

**Q: Task is overdue but I completed it**
A: Update task status to "Completed" before due date next time

**Q: Excel export is slow**
A: Reduce date range or apply filters to limit data

**Q: Dashboard numbers seem wrong**
A: Data updates every few minutes - wait and refresh page
